# Релиз v1.0.0

Это внутреннее описание релиза для команды. Здесь зафиксировано, что именно вынесено в релиз, что уже стабилизировано и какие ограничения действуют на этой ветке.

В этом релизе уже собран и стабилизирован первый рабочий срез проекта Tracker Task. Ветка отрезана от commit `509f97d008bcb445b28be1805dd4351af27770be`, защищена от прямых изменений и дальше используется только для багфиксов и блокирующих правок.

Что уже есть на этом этапе:

- зафиксирован текущий `main`;
- включена защита release-ветки;
- новых фич в релиз не добавляем;
- после freeze допускаются только критичные исправления.

Что важно для команды:

- это релизный срез, а не ветка для новой разработки;
- любые изменения после freeze допускаются только как багфиксы или блокеры;
- этот файл служит для понимания состава релиза и как база для будущих release notes;
- если в процессе появится проблема, сначала правим release-ветку, а не тащим новое в `main`.
